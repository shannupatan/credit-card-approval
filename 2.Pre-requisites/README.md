# Prerequisites

* [Python 3.x](https://www.python.org/downloads/) - Programming language used for machine learning and backend development.
* [pip Package Manager](https://pip.pypa.io/en/stable/installation/) - Used to install Python dependencies.
* [Flask](https://flask.palletsprojects.com/) - Web framework used for backend application development.
* [NumPy](https://numpy.org/install/) - Library for numerical computations.
* [Pandas](https://pandas.pydata.org/docs/getting_started/install.html) - Library for data manipulation and analysis.
* [Scikit-learn](https://scikit-learn.org/stable/install.html) - Machine learning library used for model training and evaluation.
* [Git](https://git-scm.com/downloads) - Version control system for tracking project changes.
* [GitHub](https://github.com/) - Platform for hosting and sharing repositories.
* [Visual Studio Code](https://code.visualstudio.com/download) *(Optional)* - Recommended code editor.
* [Jupyter Notebook](https://jupyter.org/install) *(Optional)* - Useful for experimentation and data analysis.
